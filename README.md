A javascript / html app in which all state, including AI keys (optional) are kept in the browser using IndexedDB. 

The App will diff a text, html or docx file and provide a visual analysis of the diff which can be printed out. 

If you wish you can add your Gemini API key and it will also  provide an AI summary of the diff changes.

You can access the App directly from: https://jimliddle.github.io/diffstack/
